IDEAaccumulation-2020-10-14

发表于 2020-10-14

一些 IDEA 的使用经验累积，尽量脱离鼠标。

快捷键

Debug

  1. debug 逐步: F8
  2. debug 进入: F7
  3. debug 继续: F9

代码书写

  1. 自动补全: Tab
  2. 通用: Alt Enter
    • 解决方案
    • 代码简化
    • 添加局部变量
    • 创建类或方法的 Javedoc 注释
    • 创建域的 Setter Getter
    • 错字
  3. 添加断点: Ctrl F8
  4. 生成代码: Alt Insert
    • 构造器
    • Getter Setter
    • JUnit 测试类
  5. 重载: Ctrl O
  6. 生成 main 函数: [main] Tab / [psvm] Tab
  7. 输出: [sout] Tab / [.sout] Tab
  8. 类型转换: [.cast] Tab
  9. 注释/取消注释当前行: Ctrl /
  10. 重构: Shift F6
  11. 代码重新排版: Ctrl Alt L
  12. 往下添加空行并转到: Ctrl Shift Enter
  13. 往上添加空行并转到: Ctrl Alt Enter

代码渲染

  1. 折叠代码块: Ctrl -
  2. 打开代码块: Ctrl +
  3. 注释渲染: Ctrl Alt Q

切换

  1. 转到定义: Ctrl B
  2. 代码文件间切换: Ctrl E
  3. 切换到 Project 视窗: Alt 1
  4. 切换到 Run 视窗: Alt 4
  5. 切换到 代码 视窗: Esc

Project 视窗

  1. 创建新事物: Alt Insert

调试运行

  1. 将当前类设置为 Main class 并运行: Ctrl Shift F10
  2. 运行: Shift F10
  3. Debug: Shift F9

其他

  1. 全局搜索: Shift Shift

设置

软换行

Soft-wrap -> ON

链式 Setter

Generate Setters -> Setter template -> Builder
