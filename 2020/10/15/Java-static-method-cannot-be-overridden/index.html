在 C++ 中，如果要在子类中重写父类的方法，需要用到关键字 virtual 和 override 。

C++ 中，只有带有 virtual 关键字的方法才能被重写。

virtual 实际上实现了 动态绑定（dynamic bind） 。是一种动态的表现。那么如果我们将这个 动态 的关键字用在 静态 关键字 staitc 上，会怎么样？答案是报错。
