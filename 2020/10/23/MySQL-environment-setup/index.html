发表于 2020-10-23

Windows 系统下的 MySQL 环境搭建。

MySQL Server

下载 MySQL Server

最新版与 5.7.32 版本，按需下载。

解压后在目录中添加必要配置文件

my.ini

  • basedir 是 MySQL 所在的绝对路径；

  • datadir 是数据存放的绝对路径；

  • 路径中要用一个斜杠 / 或者双反斜杠 \\ 来区分层次，不允许用单反斜杠。

使用 cmd 进行 MySQL 配置

  1. 使用管理员模式打开 cmd ；
  2. 进入 MySQL\bin 所在目录：
    • >cd /d E:\MySQL\mysql-5.7.32-winx64\bin
  3. 初始化 MySQL 数据地址：
    • >mysqld --initialize --console
    • --console 表示将信息打印到控制台；
    • 成功后会有初始用户 root 与其密码，请留意。
  4. 安装 MySQL :
    • >mysqld install
  5. 启动 MySQL 服务器：
    • >net start mysql
  6. 登录 MySQL 服务器：
    • >mysql -u root -p
    • 将会提示： Enter password:
    • 输入步骤 3 时获得的密码；
    • 此时进入到 mysql> 控制台。
  7. 重设 root 密码：
    • mysql>set password for root@localhost=password('newPassword');
    • 注意结尾需要分号；
    • 密码将会被设置为 newPassword 。
  • 之后用 cmd 登录 MySQL 不需要管理员模式。

MySQL Workbench

MySQL 官方提供的图形化操作工具。

下载安装 MySQL Workbench

[Download MySQL Workbench]

傻瓜式安装。

在 MySQL 服务器运行状态下启动 MySQL Workbench ，可以自动识别到，快捷连接。

更换 MySQL 路径时的坑

MySQL 会将信息注册在系统的 服务 内，在更换 MySQL 的路径时需要先将 服务 删除，然后重新搭建环境。

删除 MySQL 服务

  1. 使用管理员模式打开 cmd ；
  2. >sc delete MySQL
