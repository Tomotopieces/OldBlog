LWJGL 窗口

本篇教程中你将学会如何创建，更新，和销毁一个 LWJGL 窗口。

所有东西都是静态的

LWJGL 不像其他的 Java 窗口库，它的窗口处理是静态的。不去创建一个虚拟窗口类的实体，而是调用静态方法。

导入类

注意 Display 是 org.lwjgl.opengl.Display 的引用而不是其他的 Display 类。同样， DisplayMode 是 org.lwjgl.opengl.DisplayMode 的引用。本篇中用到的方法罗列在此。
