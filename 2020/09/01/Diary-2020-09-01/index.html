由笔试结果引发的思考

昨天，也就是刚搭好博客的同一天的晚上，笔试的邮件来了，也就点进去做了。结果很糟，满分79分，我只拿了32分···然后就想了很多，搞得彻夜难眠。

反思

  1. 有很大程度的自大。明知道自己是个自学的半吊子，明知什么都是只学了皮毛，但有一种莫名其妙的自信，就很离谱。
  2. 经常因一些没意义的事情而分散精力。搭这个博客真的有意义吗？搭博客的时间用来学其他学到一半的东西是否会更好？写这篇博文还是日记的东西的时间用来学习时候会更好？我不知道···
  3. 确实要放下一些对工作帮助不大或者没有帮助的东西，先开始学工作上需要的技能了。

欠缺

就这次笔试考察的技能而言，全都不会。

  2. Java EE
  3. Jqueury
  4. WEB
  5. Linux Bash
  6. Spring
  7. JSP
  8. Struts
  9. DBS
  10. 数据结构
  11. 算法
