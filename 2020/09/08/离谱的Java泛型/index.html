Java 的泛型是假的泛型

有个东西叫类型擦除。

这些东西是一样的。 JVM 中（也就是 Java 运行期间）是没有泛型的，全是 Object ！

所以 Java 的泛型在本质上其实就是 Object 与其子类 之间的强制类型转换。

这样也就能解释得通上面为什么一会儿报错一会儿不报错了。

所以也有人说： Java 的泛型是假的泛型。
