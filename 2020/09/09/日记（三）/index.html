感谢黑老板送的盐

老板大气，老板身体健康。

昨天晚上看了 Michael 直播的 火火老师 的独立游戏 demo 试玩，真的神仙···同样也是从 Michael 的小狐狸开始学的，八个半月做出来了 深沉之火的 domo ，太强了。

之后听火火老师在麦群里提到了 盐与避难所 这个游戏，今天在看了一下散老师的实况，哇， 2D 横板 动作 闯关 soul-like 要素齐全，很合胃口。一看 steam ， 60 块钱，好贵。在群里吐槽了一下没想到黑老板就直接送了我一个，绝了。

再次感谢黑老板！（虽然我不一定有时间玩（小声））

培训定了

确定了去培训机构了，东方锐智，不知道教学质量怎么样。叹气。

学费 18800 ， Java ，培训时间5个月，这个月下旬开班，全学完的话估计要到明年四五月份了。

也不知道博思那边到时候提供的学习内容是什么，明天得问一下。

锐智那边全日制，但周末和每天晚上肯定还是有时间的吧？晚上的时间估计也要抓起来学其他东西了。

然后我就想在这段还没去培训的时间里偷懒了， W D N M D 。不能偷懒。现在这段时间闲下来了的话就学点其他的，明明还有那么多想学的东西没学。

正式跑路

宅魂那边的活跑路了，感觉对不起宅魂，虽然有点不负责任，但我觉得也并没有亏欠他或者这个项目组什么的。情绪复杂，苦笑。

反思了一下，这个项目做了这么久，但也就前几天才刚定下来 异乡人 这个名字。策划，框架，设计，功能需求，实现，等等一万个没有想到的东西，全都一塌糊涂。 demo1 能做出来真的已经是奇迹了。

我是怎么做到现在的？因为海螺，对，因为海螺。要是海螺跑路了，那我也早就没了。

所以仔细想想，这是我第一次与团队合作来进行项目开发，虽然我负责的不是功能实现之类的代码一块的。

有学到什么吗？似乎没有。真没有···

其他收获？认识了海螺？嗯？好 gay 噢。但确实很喜欢海螺。

没了。
