Java语言中的char数据类型是否可以存放一个汉字？

  1. 可以

  2. 不可以

  3. 与运行的操作系统有关

  4. 无法判断

    正确答案：1

    Java char 采用 Unicode ，2 字节来表示一个字符。支持汉字

下列哪些选项是对方法setVar的重载（多选）

  1. private void setVar(int a, float c, int b) {}

  2. protected void setVar(int a, int b, float c) {}

  3. public int setVar(int a, float c, int b) { return a; }

  4. public int setVar(int a, int b, float c) { return a; }

  5. protected float setVar(int a, int b, float c) { return c; }

    正确答案：1 3

    重载规则:

    • 被重载的方法必须改变参数列表(参数个数或类型不一样)；
    • 被重载的方法可以改变返回类型；
    • 被重载的方法可以改变访问修饰符；
    • 被重载的方法可以声明新的或更广的检查异常；
    • 方法能够在同一个类中或者在一个子类中被重载。
    • 无法以返回值类型作为重载函数的区分标准。

switch语句中的变量可以是以下哪种类型（JDK1.6）多选：

  1. int

  2. char

  3. short

  4. string

    正确答案：1，2，3

    在jdk 1.6版本中，switch的参数无法使用String类型，只支持int,char,enum类型。
