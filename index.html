发表于 2020-10-09

今天见到了好多种快速排序···感觉和我想象中的都不一样。

Haskell 实现

先放一张我认为的快排的 Haskell 实现，简单易懂：

核心思想

我认为快速排序的核心思想是分而治之，将数组头去除后的部分以数组头为界进行分类，然后将分类后的小数组再次去头然后分类处理，以此循环，直到数组长度为 1 。

逻辑顺序

  1. 取数组首元素 key
  2. 从第二个元素开始遍历数组
  3. 比 key 小的元素放进数组 left
  4. 比 key 大的元素放进数组 right
  5. 将 left 与 right 各自排序（递归）
  6. 连接 left + key + right

              ^

String, StringBuilder, StringBuffer

发表于 2020-10-03

String

Stirng 不可变，用于创建一般来说不会多次变化的字符串。

如果某个 String 的引用的指向内容经常变化，那么就会重复开辟空间，浪费内存。

因为不会对现有的对象进行修改，所以线程安全。

StringBuilder

用于创建会被修改的字符串，不会产生新的未被使用的对象。

使用效率比 StringBuffer 高，但线程不安全。

一般情况下用这个。

StringBuffer

用于创建会被修改的字符串，不会产生新的未被使用的对象。

因为有线程锁，使用效率可能比 StringBuilder 低，但线程安全。

多线程环境下必须用这个。

错题记录-2020-09-30

发表于 2020-09-30 更新于 2020-10-03

下面不是关键字的是

  1. goto
  2. main
  3. short
  4. void

正确答案：2

goto 是关键字；
main 是标识符

设 int x = 1, y = 1; 表达式(x > 0 || y– > 1)运算后的 y 的值是

  1. 0
  2. 1
  3. 2
  4. -1

正确答案：2

|| 运算符在计算 x > 0 并判断为真后断路，不进行右边 y– > 1 的判断。

若变量 a / i 已正确赋值，合法的语句是

  1. a == 1;
  2. ++i;
  3. a = a++ = 5;
  4. a = int(i);

正确答案：2

  1. 单纯的比较结果是非法的；
  2. 正确
  3. 无法将 5 赋值给 a++ ；
  4. 强制类型转换格式错误，应为 a = (int) i；

Java 语句中运算对象必须是整形运算符的表达式是

  1. %=
  2. /
  3. =
  4. >>

正确答案：4

Java 中 %= 不一定要整数。

Java 标识符能不能是中文？

正确答案：可以是中文，但是不推荐。
