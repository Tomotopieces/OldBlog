OpenGL-Triangle

发表于 2020-09-27 更新于 2020-10-22

三角形

众所周知，三个点可以形成一个面，且 OpenGL 的图形绘制就是按各种三角形来的。所以能画一个三角形，就能画所有的东西。

VAO / VBO / EBO / Shader / Program

VAO: Vertex Array Object

顶点数组对象

用来绑定同一个图形的 VBO ， EBO 和 Program 。在绑定后，图形所有的属性调用都存在这里面，方便绘制。可以理解为一个简单的封装。

使用前记得绑定，使用后记得解绑。

VBO: Vertex Buffer Object

顶点缓冲对象

用来存一个数组，这个数组包含了一个图形的所有顶点。

假设有一三角形，顶点坐标可以为：

    vertex = 
    [
           0,  0.5, 0,
        -0.5, -0.5, 0,
         0.5, -0.5, 0
    ]

因为是 2D 的三角形，所以第三个 z 轴值忽略。

坐标值都在 -1.0 ~ 1.0 之间，叫标准化设备坐标。

EBO: Element Buffer Object

索引缓冲对象

用来存一个数组，这个数组描述了如何将图形划分为若干个三角形。

假设有一举行，顶点数组为：

    vertex = 
    [
        -0.5,  0.5, 0,
         0.5,  0.5, 0,
        -0.5, -0.5, 0,
         0.5, -0.5, 0
    ]

OpenGL 要求用户告知如何 将这个矩形拆成若干个三角形 ，再进行绘制。所以才有了 EBO 这东西，相当于给告诉 OpenGL ，请按以下顶点顺序来绘制各个三角形：

    indices =
    [
        0, 1, 2,
        2, 3, 0
    ]

这样就将矩形以左上到右下的对角线来拆分成了两个三角形。
所以 EBO 也被叫做 IBO: Indices Buffer Object 。

Shader

着色器

用来表述如何绘制。

可吞吐少量数据。

GPU 上全是这些小 Shader 。

Program

着色器程序

用来链接不同的着色器。

进行绘制时，至少需要一个 VertexShader 顶点着色器 和一个 FragmentShader 片段着色器。 Program 负责将两者链接起来。

使用时的一般流程。

在代码中使用 VAO / VBO / EBO 时，一般使用 glGenXXX 或者 glCreateXXX 函数来创建一个对象，用户用一个 unsigned int 来存放函数返回的对象 id 。在修改或调用时，也一般通过调用 glGetXXX ， glSetXXX 或者 glBindXXX 函数，传入 id 来使用。

VAO 需要绑定为 VertexArray ， VBO 和 EBO 需要在对应的 Buffer 内写入数据。

Shader 在声明创建后需要从源获取，然后编译。

Program 在声明创建后需要将 Shader 附加上来，然后链接。

OpenGL-环境搭建

发表于 2020-09-26 更新于 2020-10-22

下载必要库

  • GLFW
    • 针对 OpenGL 的 C 语言库。
    • 下载：https://www.glfw.org/download.html
    • 下载 32 位版本，因为 64 位版本不稳定。
  • GLEW
    • 用于代替 GLAD 来管理函数指针。
    • 下载：https://sourceforge.net/projects/glew/

将项目与库连接

  1. 将 GLFW 和 GLEW 解压到一个新文件夹中，文件夹名建议为 OpenGL。目录结构如下。

     - OpenGL
   - glfw-x.x.x.bin.WIN32
   - glew-x.x.x

  2. 使用 Visual Studio 创建项目，创建一个 .cpp 格式的文件，比如 main.cpp 。

  3. 设置项目属性

    1. 配置：所有配置
    2. 平台：所有平台
    3. C/C++ -> 常规 -> 附加包含内容 -> 编辑 -> 新行
      1. \OpenGL\glfw-x.x.x.bin.WIN32\include
      2. \OpenGL\glew-x.x.x\include
    4. 链接器 -> 常规 -> 附加库目录 -> 编辑 -> 新行
      1. \OpenGL\glfw-x.x.x.bin.WIN32\lib-vc20xx
      2. \OpenGL\glew-x.x.x\lib\Release\Win32
    5. 链接器 -> 输入 -> 附加依赖项 -> 编辑
      1. opengl32.lib
      2. glfw3.lib
      3. glew32s.lib
