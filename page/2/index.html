博客搭建记录

发表于 2020-08-31 更新于 2020-09-06

hexo框架搭建博客

视频参考

https://www.bilibili.com/video/BV1Yb411a7ty

创建博客

  1. Windows环境用 gitbash 来进行操作会方便很多

  2. 环境搭建

    1. nodejs 与 npm（nodejs.org）
      • 安装nodejs与npm：sudo apt install nodejs
      • 检查nodejs：node -v
      • 检查npm：npm -v
    2. cnpm（cnpm使用国内路线，可选安装）
      • 安装：npm install -g cnpm --registry=https://registry.npm.taobao.org
      • 检查：npm -v
    3. hexo
      • 安装：npm install -g hexo-cli
      • 检查：hexo -v

  3. 创建并进入博客文件路径

  4. 初始化博客：sudo hexo init

  5. 在github中创建仓库

    • 仓库名：username.github.io

  6. deployer

    • 安装：npm install --save hexo-deployer-git

  7. 修改配置文件 _config.yml

    将文件底部 deploy 部分修改为

     deploy:
     type: git
     repo: https://github.com/username/username.github.io.git
     branch: master

  8. 推送到仓库

基本操作

在博客文件夹下进行以下命令行操作

清除配置

hexo clean 或 hexo cl

重新生成

hexo generate 或 hexo g

本地挂载

hexo server 或 hexo s

推送到仓库

hexo deploy 或 hexo d

浏览器打开 localhost:4000 进行预览

换皮肤

  1. 将皮肤的仓库克隆至目录：Blog/themes/themename

  2. 修改配置文件 _config.yml

     theme: themename

  3. 清除配置

  4. 重新生成

  5. 推送到仓库

新文章

hexo new articlename 或 hexo n articlename
